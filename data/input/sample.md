## How to Change password of an Email Address in the Admin Panel

ID: `how_to_change_password_of_an_email_address_in_the_admin_panel`
Category: Admin Panel
Subcategory: How to

Alternate questions:
- How do I change password of an email address?
- How can I change password of an email address?

Answer:
To change the password, you need to follow these exact steps:
1. Login to the admin panel
2. Navigate to the user list
3. Click on the email address you want to change
4. Enter the new password and click Save

This will immediately update the user's login credentials.

Keywords: address, password, change
