## How to Create an Email Address in the Admin Panel

ID: `how_to_create_an_email_address_in_the_admin_panel`
Category: Admin Panel
Subcategory: General
Source section: How to Create an Email Address in the Admin Panel

Alternate questions:
- How do I create an email address?
- How can I create an email address?

Answer:
Steps to Create a New Email Address: Login to Admin Account: Access your Admin Email Account. Click the profile icon on the top-right corner of the screen. Select “Admin Page” from the dropdown. Navigate to Single User Section: - On the left-hand menu, click “Single User.” - Select “Add User.” Fill in User Details: A form will appear with fields to input the user’s information. Enter the required details such as: - Email ID - Domain - Mailbox Space - (Other optional details like Name, Mobile Number, etc.) Submit the Form: Once all required fields are filled, click “Create” or “Submit” to complete user creation. - Customizing the User Creation Form (Optional) If there are fields that are not relevant for your organization and you wish to make them non-mandatory: - Under the “Single User” section, go to “Customize Add User Form.” You can then select or deselect fields to set them as non-mandatory. Note: The following fields are always mandatory and cannot be made optional: - Email ID - Domain - Mailbox Space All other fields can be toggled based on your requirements.

Keywords: address, general

## How to Create Email Accounts in Bulk

ID: `how_to_create_email_accounts_in_bulk`
Category: Admin Panel
Subcategory: General
Source section: How to Create Email Accounts in Bulk

Alternate questions:
- How do I create email accounts in bulk?
- How can I create email accounts in bulk?

Answer:
Steps to Create Email Accounts in Bulk: - Login to Admin Account Log in to your Admin Email Account. Click the profile icon at the top-right corner. Select “Admin Page” from the dropdown. - Navigate to Bulk User Section - In the left-hand menu, click on “Bulk User.” - Select “Add Bulk User.” - Download and Prepare the Sample CSV File Click “Download Sample CSV” on the top-right corner. Open the CSV and fill in user data as required. ✅ Mandatory Fields: - Email ID - Mailbox Space - Domain ⚙️ Optional Fields: All other fields are optional and can be made non-mandatory via Single User → Customize Add User Form. Password: You can either mention the password of leave it blank for system to auto-generate the password - Upload the Completed CSV File Go back to Bulk User → Add Bulk User. Upload the prepared CSV file. - System Validation & Error Handling Once uploaded: If the CSV is valid, the bulk job will start automatically. If there are invalid fields or missing mandatory data, the system will reject the file. An email notification with error details will be sent to the logged-in admin email address. ✅ Action: Check your email inbox for any error notifications, correct the data, and re-upload if necessary. - Monitor Bulk Upload Status Click on the “Status” tab beside the “Sample CSV” tab. Here you can track: - Job status: Queued, Processing, Completed - Number of successful vs. failed entries - Error descriptions for failed entries

Keywords: accounts, bulk, general

Source section: How to Delete an Email Address in the Admin Panel

Alternate questions:

## How to Change password of an Email Address in the Admin Panel

ID: `how_to_change_password_of_an_email_address_in_the_admin_panel`
Category: Admin Panel
Subcategory: How to
Source section: How to Change password of an Email Address in the Admin Panel

Alternate questions:
- How do I change password of an email address?
- How can I change password of an email address?

Answer:
Steps to Change User Password: - Login to Admin Account Log in to your Admin Email account using valid credentials. Click the profile icon located at the top-right corner. - From the dropdown, select “Admin Page.” - Navigate to User Management - In the left-hand menu, click “Single User.” - Then select “User Details.” - Search for the User Use the search bar to enter the email address or username of the user whose password needs to be changed. ## How to Reset the Admin Account Password (admin@domainname)

ID: `how_to_reset_the_admin_account_password_admin_domainname`
Category: Admin Panel / Webmail
Subcategory: How To
Source section: How to Reset the Admin Account Password (admin@domainname)

Alternate questions:
- How do I reset the admin account password (admin@domainname)?
- How can I reset the admin account password (admin@domainname)?

Answer:
This SOP explains how to reset the password of the admin email account (admin@domainname) when the administrator has forgotten the password and does not have access to the current password. The reset process depends on whether the alternate email ID and mobile number are updated for the admin account. - Scenario 1: Reset Password Using “Forgot Password” Option - Condition The alternate email ID and mobile number are updated for the admin account. Steps to Reset Password: Open the webmail login page. Click on Forgot Password or use the link webmail.rediffmailpro.com/ajaxprism/forgotpwd.php. Enter the admin email address (admin@domainname). Follow the on-screen instructions to verify the account using: - Alternate email ID, or Registered mobile number. Set a new password and complete the reset process. - Scenario 2: Reset Password via Support Team - Condition The alternate email ID and mobile number are not updated for the admin account. Steps to Reset Password: Send an email request to support@rediffenterprisepro.com. The request must be sent from the registered email ID associated with the email services. Mention the following details in the email: - Domain name - Admin email address (admin@domainname) - Request for admin password reset Wait for confirmation and further instructions from the support team. For security reasons, password reset assistance is provided only after proper verification. It is strongly recommended to keep the alternate email ID and mobile number updated for the admin account to avoid delays in password recovery. -

Keywords: reset, domainname

## How to Activate/Deactivate Email Account in the Admin panel

ID: `how_to_activate_deactivate_email_account_in_the_admin_panel`
Category: Admin Panel
Subcategory: How to
Source section: How to Activate/Deactivate Email Account in the Admin panel.

Alternate questions:
- How do I activate/deactivate email account?
- How can I activate/deactivate email account?

Answer:
Steps to Activate/Deactivate an Email Account: - Login to Admin Account Log in to your Admin Email account using valid credentials. Click the profile icon on the top-right corner. Select “Admin Page” from the dropdown menu. - Navigate to User Management - In the left-hand menu, click “Single User.” - Select “User Details.” - Search for the Email Account Enter the email address or username in the search bar. Wait for the search results to populate. - Toggle Activation Status In the search result row, locate the Status column: A blue toggle indicates the user is Active. A white toggle indicates the user is Deactivated. Click the toggle to activate or deactivate the user as needed. A confirmation prompt may appear to verify your action. How to View All Deactivated Accounts: Option 1: Using MIS Reports > Deactivated Users: - From the left-hand menu, go to “MIS Reports.” Click on “Deactivated Users” to view a list of all currently deactivated email accounts. Option 2: Using MIS Reports > Download Master: - In the MIS Reports section, click on “Download Master.” Select the fields: - Status - Email ID Download the report to get the status of all active and deactivated users. Deactivated accounts cannot send emails but incoming emails are still received even for deactivated accounts. Deactivation does not delete any data from the mailbox. Use the Download Master report to track user statuses in bulk.

Keywords: activate, deactivate

## How to Give Sub-admin rights to an Email Address in the Admin Panel

ID: `how_to_give_sub_admin_rights_to_an_email_address_in_the_admin_panel`
Category: How to
Subcategory: General
Source section: How to Give Sub-admin rights to an Email Address in the Admin Panel

Alternate questions:
- How do I give sub-admin rights to an email address?
- How can I give sub-admin rights to an email address?
- How do I give admin rights to another user?
- How do I create a sub-admin?

Answer:
- Sub Category: How to Steps to Assign Admin Rights / Create a Sub-Administrator: - Login to Admin Account Log in to your Admin Email account using valid credentials. Click on the profile icon located at the top-right corner. - From the dropdown, select “Admin Page.” - Navigate to User Management - In the left-hand menu, click “Single User.” - Select “User Details.” - Search for the User Use the search bar to enter the email address or username of the user you want to assign admin rights to. Wait for the results to populate. - Assign Admin Rights (Option 1: Via Admin Rights Icon) In the user row on the right side, click the Admin Rights icon (🛡️). A panel will appear with the option to assign: - Full admin rights (access to all features), or Limited admin rights (e.g., only permission to add users). Select the permissions based on the required role and confirm. - Assign Admin Rights (Option 2: Via Edit Panel) Alternatively, click the Edit icon (✏️) on the user row. A side panel will open from the right-hand side. - Navigate to the fifth tab labeled “Admin Rights.” Choose between: - Full Admin Access, or Custom Permissions (e.g., only add users, no delete/change password access). Save changes to apply the assigned rights. Examples of Sub-Administrator Roles: User1 → Sub-admin with only add user rights (Cannot delete or change user passwords) User2 → Sub-admin with add and delete user rights (Cannot change passwords but can manage user accounts) Additional Notes: You can create multiple sub-administrators with varied permissions as per your organizational requirements. Sub-admins will only have access to the features specifically assigned to them. Changes in permissions can be updated at any time using the same steps.

Keywords: give, sub-admin, rights, address, general

Alternate questions:
- How do I assign additional space to an email account?
- How can I assign additional space to an email account?

Answer:
You can manually allocate additional mailbox space to a user from the Manage Space section. - https://support.rediffmailpro.com/article/space-management-process-automatic-manual/ Steps to Add Space to a User: - Login to Admin Panel - Go to Single User → Manage Space - Select Manual Mode - Use the “Add space for a user” option - Enter User Details - Enter User ID - Select the Domain - Click Submit - View User Space Details - Assigned mailbox size - Total space allocated - Space consumed - Enter Space to Add - Specify the required space (in MB) Choose Allocation Method: - Additional Storage Purchased → Uses extra purchased storage From Existing Users → The system will automatically identify users who have unused space and will deduct the required amount from them Reshuffle Space → a list of all users will appear. You can manually select one or more users who will donate the space.
