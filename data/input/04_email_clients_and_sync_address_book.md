## How to Configure Calendar Sync Using Outlook Plugin

ID: `how_to_configure_calendar_sync_using_outlook_plugin`
Category: Email Client
Subcategory: How To
Source section: How to Configure Calendar Sync Using Outlook Plugin

Alternate questions:
- How do I configure calendar sync using outlook plugin?
- How can I configure calendar sync using outlook plugin?

Answer:
This SOP explains how to enable Calendar Sync between Rediffmail Enterprise and Microsoft Outlook by installing the Calendar Plugin Installer for Outlook. Once configured successfully, users will be able to view and manage their webmail calendar events directly from Outlook. - Prerequisites Before installing the calendar plugin, ensure the following: The setup file is downloaded on the desktop. The system has full administrator rights. Microsoft Outlook is closed before installation. Stable internet connectivity is available. Steps to Configure Calendar Sync: Step 1: Download Calendar Plugin: Download the Calendar Plugin Installer using the link below: - http://beta.rediffmailpro.com/calp/bs/setup.exe Step 2: Save the Installer: Ensure the downloaded setup file is saved on the desktop. Step 3: Install the Plugin: Right-click on the setup file and select Run as Administrator. Follow the on-screen installation instructions. Step 4: Restart Outlook: Once the installation is completed: Exit Microsoft Outlook completely. Reopen Outlook. Step 5: Verify Calendar Sync: After restarting Outlook, check for the Calendar tab. The presence of the calendar tab confirms successful installation. The plugin must be installed with administrator privileges to function correctly. If the Calendar tab does not appear: Restart the system and recheck. Ensure Outlook version compatibility. Any antivirus or endpoint security software may block installation and should be temporarily disabled if required. Calendar synchronization may take a few minutes after first login.

Keywords: calendar, sync, outlook, plugin

## What is the difference between the Personal Address Book and the Global Address Book in Rediffmail Enterprise?

ID: `what_is_the_difference_between_the_personal_address_book_and_the_global_address_book_in_re`
Category: Contacts
Subcategory: Address Book
Source section: Rediffmail Enterprise – Address Book

Alternate questions:
- How do I the difference between the personal address book and the global address book in rediffmail enterprise?
- How can I the difference between the personal address book and the global address book in rediffmail enterprise?

Answer:
Rediffmail Enterprise – Address Book. Like all email service providers, Rediffmail Enterprise too has 2 types of address book. 1. Personal Address Book - Managed by individual users - Contains personal or frequently used contacts - Visible only to the user who created it 2. Global Address Book (GAL) - Managed by the organization’s email administrator Contains official email addresses of: - Employees - Shared mailboxes - Groups / distribution lists - Visible to all users in the organization The Global Address Book ensures everyone can easily find and email internal contacts.
